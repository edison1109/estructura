Describa las principales etiquetas HTML

ETIQUETA DESCRIPCIÓN
HTML Indica que le documento en cuestión está constituido con dicho lenguaje.
HEAD Cabecera que contiene básicamente la información destinada a los navegadores que queda oculta al usuario.
BODY Llamado el cuerpo del documento aquí va lo que se le muestra al usuario.
META Se utilizan en la cabecera, son de gran utilidad para los navegadores u otros programas que pueden velarse de esta información.
TITLE La etiqueta de título es un elemento meta del texto fuente del sitio web que no es visible, y el cual define el título en sí de la página.
FONT Fuente, indica el tamaño, color, o forma del texto que el contiene.
BR Genera un salto de línea, espacio entre textos.
ETIQUETA P Es el encargado de distribuir el texto en párrafos.
DIV División, sirve para crear secciones o agrupar contenidos.
UL, OL, DL, LI Etiquetas que hacen parte de la creación de lista sean ordenada, no ordenas o listas de definiciones.
HR Es para introducir una línea o raya Horizontal.
A Hiperenlace, vincula un documento activo con otro.

Describa las principales etiquetas CSS

ETIQUETA DESCRIPCIÓN
COLORS La manera de dar color y estilo a los textos, fondos, bordes, etc. Cada color se puede representar de manera diferente. Ejem: Blue,#0000ff, #00f, rgb(0,0,255)
BACKGROUNDS Las propiedades del fondo se utilizan para definir los efectos como Color de Fondo, Imagen de Fondo, Adjunto de Fondo, Posición de Fondo, Repetición de Fondo.
BORDERS Permite especificar el estilo, el ancho y el color del borde de un elemento.
MARGINS Se utiliza para crear un espacio alrededor de los elementos, fuera de los bordes definidos, tiene control sobre todas las márgenes ancho, color y estilo.
TEXT Se usa para dar forma al texto como alineación, transformación y color.
FONTS Las propiedades de fuente definen la familia de fuente, la negrilla, el tamaño y el estilo del texto.
HEIGHT Especifica la altura del área del contenido de un elemento, ésta se encuentra deltro del Padding, Bordes y Margen del elemento.
WIDTH Establece el ancho del elemento.
LINKS En CSS los enlaces se pueden diseñar de diferentes maneras dándole estilo y color.
TABLES Las tablas creadas en HTML se pueden mejorar mucho en CSS dándole borde, altura, ancho y color, rellenar y sombrear.
POSITION Especifica como un elemento es posicionado en el documento, las propiedades Top, Right,Botton y Left determinan la ubicación final.
OVERFLOW Especifica qué debería suceder si el contenido desborda el cuadro de un elemento, si se debe recortar contenido o agregar barra de desplazamiento.
FLOAT Especifica cómo debe flotar un elemento y el resto fluirá a su alrededor.
